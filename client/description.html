The following steps are required to run the Electre Tri-C method correctly:


  1. Define criteria set.
    • Mandatory set: name, direction, weight, indifference, preference and veto;
    • Weights can be defined: added manually or using the method "determining weights of criteria".
  2. Define alternatives set and fill the performance table.
  3. In Decision Configurations:
    1. Define categories;
    2. Give a value for the Credibility Lambda ( λ ) between [0.5 ; 1.0];
    3. Fill the Profiles Performance Table.
